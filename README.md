# Sun and moon migration-demo

🤔 Why?
---------
Demonstrate how to setup database migration using SQLalchemy and alembic.

🛠️ Requirements
----------------
See the requirements.txt file.
- Python >= 3.9
- sqlalchemy
- alembic
- fastapi
- pydantic
- uvicorn
